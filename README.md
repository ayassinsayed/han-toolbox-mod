# han-toolbox-mod
**Mod for HAN ToolBox.**

![Alt-psnstuff](https://cdn1.savepice.ru/uploads/2018/4/1/618afeb1e624b7de7be174fa7fe983e6-full.jpg) 

Allow download game from psnstuff.

But you still need install act.dat and *.rif to activate game

Recommend LAN connection

## Usage:
	Just install pkg (need HAN). For icon update recource
	You can update base from ps3 - update Toolbox
	
## Использование:	
	Просто установите pkg (нужен HAN), для иконок обновите ресурсы к HAN ToolBox.
	Всегда свежую базу можно скачать через PS3: обновить ToolBox в "дополнительных возможностях - опции toolbox"
